# Condicional `se`

A estrutura condicional `se` executa um bloco de código apenas se uma expressão condicional for verdadeira. A sintaxe é a seguinte:

```lina
se expressao repetir
    # corpo condicional
fim
```

### Exemplo

```lina
inteiro x := 5
se x > 0 repetir
    # código a ser executado se x for maior que 0
fim
```
