# Extruturas de Controle

## Laço `para`

O laço `para` é uma estrutura especializada em iterar em inteiros, sua sintaxe é definida a seguir:

```lina
para [variavel] ate [limite] repetir
    # corpo do laço
fim
```

### Exemplo

```lina
programa Exemplo

para i ate 10 repetir
    saida("i = ", i)
fim
```
